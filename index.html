This isn't about speed. Always, always correct mistakes. Make 100% accuracy you goal. By correcting all errors you can still score 100%. This program will allow you to make 5 consecutive errors (very unlikely to happen). That is your last chance to use Backspace and type those letters again. If you fail to do so after the fifth wrong character in a row, the app will assume that you are doing it on purpose and after a goodbye message the page will reload and current session will be lost, including all control settings. Then you can start a new session. Type away player!
