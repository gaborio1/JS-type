How to use this app

It isn't about speed. Always, always correct mistakes. Make 100% accuracy you goal. By correcting all errors you can still score 100%. This program will allow you to make 5 consecutive errors. That is your last chance to use Backspace and type those letters again. If you fail to do so after the fifth wrong character in a row, the page will reload and current session will be lost, including all control settings.


Controls

  • Difficulty: Most common 100 / 200 English words and JavaScript reserved words.
  • Line length: Sets the average length of text that fits into a text field (Ranges from 4-6 to 11-13 words).
  • Punctuation and Capital letters: On / Off. Affects every word.
  • Timer: 1 minute (default) / Off.
  • Sound: On / Off. Keyboard click for valid keys and alert for wrong keys.
  • Info: Show / hide info.

Text fields

  • Active text field: Text field 1. Words currently being typed, cursor tracks current position.
  • Next field: Text field 2. Preview of the next active field text.

Input section

  • Timer: 60 second countdown (active by default) On / Off.
  • Text input: User input
  • Start button: Generates text

Counters / Statistics

  • Average speed: Shows current speed during timer and average speed at the end of session
  • Green / Red / Backspace: Counter for each colour (correct key, wrong key, backspace)
  • Keystrokes: Total number of keystrokes (including space / backspace)
  • Accuracy: Current and total accuracy percentage rounded down to nearest integer

Key / Cursor colour codes

  • Correct: Green.
  • Wrong: Red.
  • Backspace: orange.
  • Not typed: Characters at / ahead of cursor position.
  • Skipped: Space typed anywhere on word.
